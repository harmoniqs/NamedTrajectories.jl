Parameters

NamedTrajectory.jl support passing parameters as a Tuple when construct a NamedTrajectory.

using NamedTrajectories

First we need to define number of timesteps and timestep

T = 10
dt = 0.1
0.1

then build named tuple of components and data matrices.

components = (
    x = rand(3, T),
    u = rand(2, T),
    Δt = fill(dt, 1, T),
)
(x = [0.49198258960223185 0.25242412346513377 … 0.16600100615893165 0.8046271583022492; 0.45130336156683615 0.6385103435817046 … 0.17802015482248323 0.08363057787592898; 0.6471606150752865 0.582880950848772 … 0.8167278766088383 0.3305516461533047], u = [0.068424974835961 0.7560964684876124 … 0.008269648941149566 0.5198973463574404; 0.2156856479499515 0.8428501961114259 … 0.2084418510330942 0.6971318289841089], Δt = [0.1 0.1 … 0.1 0.1])

we must specify a timestep and control variable for the trajectory.

timestep = 0.1
control = :u
:u

some global params as a NamedTuple

params = (
    α = rand(1),
    β = rand(1)
)
(α = [0.42953689109113546], β = [0.27091339314714])

we can now create a NamedTrajectory object with parameters specification.

traj = NamedTrajectory(components; timestep=timestep, controls=control, global_data=params)
