Parameters

NamedTrajectory.jl support passing parameters as a Tuple when construct a NamedTrajectory.

using NamedTrajectories

First we need to define number of timesteps and timestep

T = 10
dt = 0.1
0.1

then build named tuple of components and data matrices.

components = (
    x = rand(3, T),
    u = rand(2, T),
    Δt = fill(dt, 1, T),
)
(x = [0.9315237654632148 0.36564450633859724 … 0.19930618491907814 0.26228432762587517; 0.9763863454793245 0.46984202439739264 … 0.7249563654129593 0.1334209201682771; 0.7027446136251727 0.05027719999737712 … 0.43437551911976835 0.26376969644550086], u = [0.9882081170747774 0.030467984458576458 … 0.6811677379613739 0.16223968516422216; 0.8503433715169281 0.5087377020596056 … 0.7865136091391733 0.5723162865864104], Δt = [0.1 0.1 … 0.1 0.1])

we must specify a timestep and control variable for the trajectory.

timestep = 0.1
control = :u
:u

some global params as a NamedTuple

params = (
    α = rand(1),
    β = rand(1)
)
(α = [0.8903258373722869], β = [0.5576781643007569])

we can now create a NamedTrajectory object with parameters specification.

traj = NamedTrajectory(components; timestep=timestep, controls=control, global_data=params)
