Parameters

NamedTrajectory.jl support passing parameters as a Tuple when construct a NamedTrajectory.

using NamedTrajectories

First we need to define number of timesteps and timestep

T = 10
dt = 0.1
0.1

then build named tuple of components and data matrices.

components = (
    x = rand(3, T),
    u = rand(2, T),
    Δt = fill(dt, 1, T),
)
(x = [0.14491076107723622 0.44466211582240056 … 0.9986648506806319 0.5585686612263188; 0.6100674675499009 0.9380403686913694 … 0.3372755277756827 0.22548722680175615; 0.8043461327313108 0.6250039540402381 … 0.00650713877549991 0.18330811008882397], u = [0.8889567189801602 0.20828592664490453 … 0.036344441564295726 0.43226506694906697; 0.6102590389818117 0.5031297906425193 … 0.2814346103252692 0.37528386936027447], Δt = [0.1 0.1 … 0.1 0.1])

we must specify a timestep and control variable for the trajectory.

timestep = 0.1
control = :u
:u

some global params as a NamedTuple

params = (
    α = rand(1),
    β = rand(1)
)
(α = [0.3980128202298193], β = [0.18319674459561464])

we can now create a NamedTrajectory object with parameters specification.

traj = NamedTrajectory(components; timestep=timestep, controls=control, global_data=params)
