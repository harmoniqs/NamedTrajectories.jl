Parameters

NamedTrajectory.jl support passing parameters as a Tuple when construct a NamedTrajectory.

using NamedTrajectories

First we need to define number of timesteps and timestep

T = 10
dt = 0.1
0.1

then build named tuple of components and data matrices.

components = (
    x = rand(3, T),
    u = rand(2, T),
    Δt = fill(dt, 1, T),
)
(x = [0.21197630234124043 0.7251131510363202 … 0.0800057895232873 0.6048999411160882; 0.4076779827699627 0.555498880526899 … 0.8628675527979174 0.5186649203259937; 0.589824922453843 0.0658137258562973 … 0.4237928853452525 0.0072166038354771755], u = [0.8348680539645466 0.5699284638600937 … 0.48064123703508854 0.8135542505431727; 0.6396560684039851 0.21161620560647476 … 0.5225748541180596 0.7349479744994878], Δt = [0.1 0.1 … 0.1 0.1])

we must specify a timestep and control variable for the trajectory.

timestep = 0.1
control = :u
:u

some global params as a NamedTuple

params = (
    α = rand(1),
    β = rand(1)
)
(α = [0.44015640123973565], β = [0.32279661060077536])

we can now create a NamedTrajectory object with parameters specification.

traj = NamedTrajectory(components; timestep=timestep, controls=control, global_data=params)
