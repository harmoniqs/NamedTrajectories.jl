Parameters

NamedTrajectory.jl support passing parameters as a Tuple when construct a NamedTrajectory.

using NamedTrajectories

First we need to define number of timesteps and timestep

T = 10
dt = 0.1
0.1

then build named tuple of components and data matrices.

components = (
    x = rand(3, T),
    u = rand(2, T),
    Δt = fill(dt, 1, T),
)
(x = [0.6167904194239578 0.018093534835558645 … 0.9932337005553429 0.07146468962815178; 0.014928983884374913 0.14567928027333954 … 0.5796443735069254 0.9488502037848944; 0.9529815666204897 0.20451684285058813 … 0.06722542005549081 0.08949672839960887], u = [0.6075693990148141 0.4273481729494659 … 0.593243888932071 0.6159546880768856; 0.8453963681512746 0.6834375094786701 … 0.9643139195668242 0.5514906987916401], Δt = [0.1 0.1 … 0.1 0.1])

we must specify a timestep and control variable for the trajectory.

timestep = :Δt
control = :u
:u

some global params as a NamedTuple

params = (
    α = rand(1),
    β = rand(1)
)
(α = [0.42668731875498833], β = [0.858146492653087])

we can now create a NamedTrajectory object with parameters specification.

traj = NamedTrajectory(components, params; timestep=timestep, controls=control)
T = 10, (x = 1:3, u = 4:5, → Δt = 6:6), (α = 1:1, β = 2:2)
